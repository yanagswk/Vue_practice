# 機能概要  
 - todoリスト新規追加機能  
 - todoリスト削除機能  
 - todoリスト集計機能  
 - todoリストドラック&ドロップ機能  
 - タスク追加機能  
 - タスク削除機能  
 - タスク編集機能  
 - タスク工数機能  
 - タスクドラック&ドロップ機能  

# 使用技術  
 - Vue.js 2.6.11  
 - Vue CLI 4.1.11  
 - Vuex 3.4.0  
 - Draggable 2.23.0
