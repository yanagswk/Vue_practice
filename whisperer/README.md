# whisperer
twitterのような呟きができるアプリ  
<img width="1200" alt="スクリーンショット 2021-05-05 15 41 44" src="https://user-images.githubusercontent.com/61370785/117105351-cffd3e00-adb8-11eb-83a8-a46c4da9aedf.png">  


# 機能概要  
 - googleアカウントからのログイン機能  
 - firebaseとの連携
 - メッセージ追加機能  
 - メッセージ編集機能  
 - メッセージ削除機能  
 - メッセージ一蘭機能  
 - メッセージ集計機能  
 - マイページ編集機能  
 - モーダルウィンドウ  

# 使用技術  
 - vue.js 2.6.11  
 - vue CLI 4.1.11  
 - vue-router 3.2.0  
 - vue2-filters 0.12.0  
 - vuefire 3.0.0
 - firebase
